# US Flights Data Analysis (2008)

## Project Overview
This project analyzes the 2008 US Flights dataset containing more than 7 million flight records. The goal was to explore airline performance, flight delays, cancellations, route popularity, and temporal patterns using Python data analysis libraries.

The project demonstrates practical data analytics skills including data cleaning, aggregation, statistical analysis, business insight generation, and data visualization.

## Dataset
The dataset contains information about:
* Airlines and flight numbers
* Departure and arrival airports
* Flight distances
* Departure and arrival delays
* Cancellation reasons
* Flight schedules
* Weather and operational disruptions

Dataset size: **7+ million flight records**
